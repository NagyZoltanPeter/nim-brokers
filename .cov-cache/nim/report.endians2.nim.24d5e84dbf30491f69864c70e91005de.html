121 ## Read bytes and convert to an integer according to the given endianness.
122 ##
123 ## Note: The default value of `system.cpuEndian` is not portable across
124 ## machines.
125 ##
126 ## Panics when `x.len < sizeof(T)` - for shorter buffers, copy the data to
127 ## an `array` first using `arrayops.initCopyFrom`, taking care to zero-fill
128 ## at the right end - usually the beginning for big endian and the end for
129 ## little endian, but this depends on the serialization of the bytes.
130
131 # This check gets optimized away when the compiler can prove that the length
132 # is large enough - passing in an `array` or using a construct like
133 # ` toOpenArray(pos, pos + sizeof(T) - 1)` are two ways that this happens
134 doAssert x.len >= sizeof(T), "Not enough bytes for endian conversion"

137 for i in 0..<sizeof(result):
138 result = result or (T(x[i]) shl (i * 8))
139 else:
140 # `copyMem` helps compilers optimize the copy into a single instruction, when
141 # alignment etc permits
142 copyMem(addr result, unsafeAddr x[0], sizeof(result))
