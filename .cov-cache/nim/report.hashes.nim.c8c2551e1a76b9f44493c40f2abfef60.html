10 ## This module implements efficient computations of hash values for diverse
11 ## Nim types. All the procs are based on these two building blocks:
12 ## - `!& proc <#!&,Hash,int>`_ used to start or mix a hash value, and
13 ## - `!$ proc <#!$,Hash>`_ used to finish the hash value.
14 ##
15 ## If you want to implement hash procs for your custom types,
16 ## you will end up writing the following kind of skeleton of code:
17
18 runnableExamples:
19 type
20 Something = object
21 foo: int
22 bar: string
23
24 iterator items(x: Something): Hash =
25 yield hash(x.foo)
26 yield hash(x.bar)

28 proc hash(x: Something): Hash =
29 ## Computes a Hash from `x`.
30 var h: Hash = 0
31 # Iterate over parts of `x`.
32 for xAtom in x:
33 # Mix the atom with the partial hash.
34 h = h !& xAtom
35 # Finish the hash.
36 result = !$h
37
38 ## If your custom types contain fields for which there already is a `hash` proc,
39 ## you can simply hash together the hash values of the individual fields:
40
41 runnableExamples:
42 type
43 Something = object
44 foo: int
45 bar: string
46
47 proc hash(x: Something): Hash =
48 ## Computes a Hash from `x`.
49 var h: Hash = 0
50 h = h !& hash(x.foo)
51 h = h !& hash(x.bar)
52 result = !$h
53
54 ## .. important:: Use `-d:nimPreviewHashRef` to
55 ## enable hashing `ref`s. It is expected that this behavior
56 ## becomes the new default in upcoming versions.
57 ##
58 ## .. note:: If the type has a `==` operator, the following must hold:
59 ## If two values compare equal, their hashes must also be equal.
60 ##
61 ## See also
